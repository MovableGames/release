# MovableGames Release

Welcome to the MovableGames release repository. This repository is used to distribute and manage releases of the **MovableGames Desktop App and Platform**.

## About MovableGames

MovableGames is an innovative educational platform that combines:
- 🎮 **Adaptive game-based learning** with proprietary AI tutoring
- 💻 **Desktop and cloud-based applications** for seamless learning experiences
- 📊 **Advanced analytics** including career prediction and performance tracking
- 🏫 **Curriculum alignment tools** for institutional and individual learners

## Repository Purpose

This repository serves as the official release distribution center for:
- **Desktop App Installers** - Available for download and automated installation
- **Release Notes** - Detailed information about new features, improvements, and bug fixes
- **Documentation** - Setup guides and platform overview

## Getting Started

To get the latest version of the MovableGames Desktop App:

1. Visit the [Releases](../../releases) page to find the latest version
2. Download the appropriate installer for your operating system
3. Follow the installation instructions in the release notes
4. Create or log into your MovableGames account to get started

## System Requirements

- **Windows** 11 or later
- Stable internet connection for cloud services synchronization
- Minimum 4GB RAM recommended
